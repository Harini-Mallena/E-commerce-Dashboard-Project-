# 📊 E-Commerce Sales Dashboard

## 🔹 Overview
An interactive E-Commerce Dashboard created in Microsoft Excel to analyze sales performance, product insights, and customer behavior using KPIs, charts, and dynamic slicers.

## 🔹 Dashboards Included
- Sales Performance Dashboard
- Product Analysis Dashboard
- Customer Analysis Dashboard

## 🔹 Features
- KPI Tracking
- Dynamic Slicers
- Monthly Sales Trends
- Region-wise Analysis
- Product Profitability Analysis
- Customer Insights

## 🔹 Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Visualization

## 🔹 Skills Demonstrated
- Dashboard Designing
- KPI Analysis
- Data Visualization
- Business Storytelling
- Data Analysis

## 🔹 Outcome
This project helps analyze business performance and supports data-driven decision-making through interactive visual insights.
